Introduction

Purpose

Nature v1.0 is an inventory management system for the dairy farming market segment. The Assets feature will consist of a user credential protected UI that interacts with a service that manages user registration, asset Nature will consist of next generation AMP collection of tools for managing social media advertising. The Onboard feature will be a UI that interacts with a service that manages mappings between Agencies, Users, Partners and Initiatives enabling registration and management of all publishers from the same tool. This is similar to the earlier AMP feature: Client Onboarding.
